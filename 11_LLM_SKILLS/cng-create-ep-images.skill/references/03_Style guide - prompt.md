# Workplace Field Notes --- Style Guide

## Purpose

Create editorial illustrations for the Russian Substack and podcast
**«Коллеги, не ребята»**.

This is **not** a poster, thumbnail, or marketing illustration.

Each image is one page from an ongoing visual notebook called
**Workplace Field Notes**.

## Visual Identity

Imagine a hybrid of: - storyboard - detective investigation board -
product designer's notebook - UX workshop whiteboard - organizational
anthropology field journal

Someone is investigating how organizations actually work.

## Physicality

Everything must feel physically real: - open notebook - loose paper -
index cards - sticky notes - masking tape - paper clips - thick
markers - pen and pencil - folded corners - realistic shadows

Photographed from above with a slight angle.

## Drawing Style

Hand drawn. Visible marker strokes. Rough arrows. Notebook annotations.
Crossed-out ideas. Construction lines. Hatching. Slight imperfections.

Never polished. Never vector. Never corporate.

## Color

Warm off-white paper. Charcoal black ink. Soft gray shadows.

Exactly one accent color: - rust red - burnt orange - muted mustard -
muted olive - dusty blue

## Composition

The center must remain visually quiet because a Play button will overlay
it, but it must stay unmarked. Do not draw a visible oval, circle,
target mark, play-button placeholder, empty outline, or reserved-area
guide in the center.

Distribute the composition around the four corners.

Use only 3--6 major objects.

Leave generous negative space.

## Text

All visible handwriting must be in Russian.

Use only 1--3 word handwritten labels.

No titles. No branding. No logos.

## Recurring Elements

-   open notebook
-   two markers in the lower left
-   episode card in the upper left
-   sticky note in the upper right
-   investigation card in the lower right
-   one episode-specific diagram
-   masking tape
-   paper texture
-   realistic shadows

Every illustration should feel like another page from the same notebook.
